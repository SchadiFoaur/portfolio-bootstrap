The beginning - First Entry

January 7, 2022

This was my first attempt at creating a website. I followed the tutorial offered by Code makery step by step. All and all it wasn't that hard. I did however have some trouble figuring out how to use Bootstrap. But the biggest challenge was the animated GIF. On the homepage i wanted to include an animation of my self instead of a normal image like in the tutorial. The problem is that i have no skills in working with images let alone animations. Then suddenly i remembered how on iphones and Androids you could text someone an emoji that looks like you, so i turned my picture to an animated emoji using Samsung AR on my phone and used it. Currently i'm trying to get rid of the borders around the animation.
